2.jdk jre jvm 的联系
JDK  =  JRE + JAVA 开发工具（Java Javac Javadoc Javap）
JRE  =  jVM + java的核心类库


3.配置环境变量作用

控制台输入java指令后先在所在的文件目录寻找文件 若无 则在path的环境变量的目录中寻找 

所以新增JAVA_HOME环境变量 所指向的目录为jdk的文件目录 然后再在path环境变量目录中添加%JAVA_HOME%\bin

从而找到jdk的bin文件

pash->%JAVA_HOME%->jdk所在目录->jdk\bin
